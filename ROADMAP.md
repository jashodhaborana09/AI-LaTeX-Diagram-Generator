# Roadmap

## Near Term

- Add optional object storage for generated PDFs, PNGs, and TEX files.
- Add Redis-backed rate limiting for multi-instance deployments.
- Add frontend screenshot assets for release documentation.
- Add optional cleanup jobs for generated artifacts.

## Mid Term

- Add authenticated workspaces.
- Add saved diagram history backed by durable storage.
- Add richer diagram-specific validation for UML, sequence, network, and architecture diagrams.
- Add provider-specific deployment templates for common cloud platforms.

## Long Term

- Add collaboration workflows for shared diagram editing.
- Add export bundles with metadata and source prompts.
- Add observability integrations for logs, metrics, and traces.
- Add optional background job processing for long compilation workloads.
